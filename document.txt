This Laravel 12 application provides a basic API for managing users.

## ðŸ”— API Endpoints

### 1. Get All Users

- **Method**: GET  
- **URL**: `/api/users`  
- **Response**: JSON list of all users

```json
[
  {
    "id": 1,
    "name": "Amit Sharma",
    "email": "amit@example.com"
  },
  ...
]
```

---

### 2. Get User by ID

- **Method**: GET  
- **URL**: `/api/users/{id}`  
- **Response**: JSON of the user or 404 if not found

```json
{
  "id": 1,
  "name": "Amit Sharma",
  "email": "amit@example.com"
}
```
